# 计算机组成原理

## 绪论

### 计算机发展历程

计算机系统 = 硬件 + 软件。

#### 软件

##### 软件的分类

- **系统软件 - 用来管理整个计算机的系统**
  - 操作系统、数据库管理系统（DBMS）、标准程序库、网络软件、语言处理程序、服务程序。
- **应用软件 - 按任务需要编制成的各种程序**

##### 软件发展历程

- 微程序
- 机器语言
- 操作系统
- 汇编语言
- 高级语言

#### 硬件

电子计算机之父 ——冯诺依曼

电子管 -> 晶体管 -> 中小规模集成电路 -> 大、超大规模集成电路
